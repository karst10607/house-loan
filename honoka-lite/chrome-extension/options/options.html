Honoka Bridge Setup

The bridge is a tiny Node.js server that lets Honoka save docs, preview, and diff locally. Replace <repo-root> with the path to your cloned repo.

1
Open Terminal and run this (replace <repo-root>):
cd <repo-root> && node honoka-bridge/index.js
2
Done. Keep the terminal open. The dot above should turn green.
For auto-start on login, run bash <repo-root>/honoka-bridge/install.sh instead.
